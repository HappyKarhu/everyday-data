Why Everyday Data? #

Small everyday actions often feel insignificant.

This project examines whether they actually make a measurable difference using public datasets, scientific literature, and transparent calculations.

Every research project follows the same methodology:

  • 📊 Official data sources
  • 🐍 Python (Pandas) analysis
  • 📈 Clear visualizations
  • 🔍 Transparent assumptions
  • 📚 Scientific references
